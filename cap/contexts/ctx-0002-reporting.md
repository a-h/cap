# Reporting

## Description

The tools that read a loaded Model (con-0004) to guide development: listing entities,
showing one entity's links, building a Graph (con-0005) of the composition, providing
an agent the Capability bundle (con-0006) it needs to develop a feature, and assembling
a Review packet (con-0007). Reporting depends on the Documentation model context
(ctx-0001) for the entities and references it navigates; it neither defines the model's
shape nor writes to it.
